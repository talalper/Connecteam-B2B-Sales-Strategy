# B2B Tech Sales Strategy: Connecteam
Project Overview
This project outlines a strategic B2B sales approach for Connecteam, a leading all-in-one mobile solution for deskless workforces.
The assignment focuses on a targeted sales pitch to Afikim Electric Vehicles, addressing their unique operational challenges.

Target Client: Afikim Electric Vehicles
* Industry: Electric Vehicle Manufacturing & Services.
* Challenges Identified: Scattered communication (WhatsApp/Excel), complex shift scheduling for field staff, and manual payroll processing leading to friction.
* Proposed Solution: Implementing Connecteam's unified hub for communication, automated scheduling, and GPS-enabled time tracking.
